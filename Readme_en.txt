Information
-------------------------------------------------------------------------------
Marimont’s algorithm can detect the existence or not of a circuit in a graph. 
This project is a readjustment of the first implementation of this algorithm taking
into account this time component domain design software architecture, a concept that
makes software scalable and maintainable, thanks to the transformation of the business
layer and presentation of the software, replaceable DLL libraries on the fly.
 
     * Enter the list of vertices separated by semicolons (A, B, C)
     * Enter the list of arcs separated by semicolons ((A, B); (B, C); (C, A))



Running
-------------------------------------------------------------------------------
Unzip the Run.zip and launch the Marimont.exe

-------------------------------------------------------------------------------
